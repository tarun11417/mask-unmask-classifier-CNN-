# Mask-Unmask Classifier (CNN)

## 📌 Project Overview
This project uses a Convolutional Neural Network (CNN) to classify images into:
- With Mask 😷
- Without Mask ❌

## ⚙️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

## 📂 Dataset
Dataset contains images of people with and without masks.
